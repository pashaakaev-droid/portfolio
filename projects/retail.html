Демонстрационные проекты

Пресейловые активности РАМАКС

После ухода SAP с российского рынка компания начала переход на отечественные BI-решения. Команда аналитики изучала продукты на рынке и тестировала их в рамках пилотных проектов. Одним из выбранных инструментов стала платформа Visiology.

После прохождения вендорского обучения было принято решение реализовать демонстрационный BI-кейс для ритейла.

Цель проекта

Разработать демонстрационный BI-кейс для ритейла, показывающий возможности платформы Visiology и подход к сквозному анализу бизнес-показателей.

Роль в проекте

В рамках проекта я отвечал за подготовку данных и BI-моделирование, разработку дашбордов в Visiology и реализацию сквозного аналитического сценария, а также за демонстрацию решений

Кейс

В следующих блоках показан функционал дашбордов:

Продажи

Дашборд предназначен для анализа продаж и выручки с возможностью детализации по периодам, географии и категориям товаров.

Реализованные возможности:

- Агрегированные KPI по заказам и выручке с динамикой к предыдущему периоду или сезону

- Drill-down в распределениях с вынесением непопулярных значений в «Прочее»

- Кастомный treemap с цветовой индикацией отклонений выручки

- Многоуровневая детализация динамики продаж (год → месяц → категории) и передача контекста и фильтров между виджетами и глобальными фильтрами

- Переход на дашборд "Основные показатели" через Treemap с передачей фильтров

Основные показатели

Дашборд второго уровня для мониторинга проблемных заказов и выявления причин задержек.

Реализованные возможности:

- Интерактивные диаграммы с передачей значений в глобальные фильтры и контекстная перестройка виджетов по бизнес-правилам задержки доставки

- Детализация заказов в интерактивной таблице

- Кастомное условное форматирование и визуальная индикация статусов

- Переход на дашборд «Детализация заказа с передачей контекста и фильтров

- Переход на дашборд «Логистика» при клике на задержанные заказы с передачей глобальных фильтров

Детализация заказа

Этот дашборд 3-го уровня предназначен для детального анализа конкретного заказа и входящих в него товаров.

Реализованные возможности:

- Отображение ключевой информации по заказу (номер, дата)

- Анализ параметров доставки: перевозчик, стоимость, плановая и фактическая даты

- Визуализация отклонений по срокам доставки с подсказками План / Факт

- Детализация состава заказа в интерактивной таблице

- Фильтрация данных и синхронизация блоков по выбранному товару

- Динамическое обновление описания товара, изображения и рекомендованной цены

Логистика

Дашборд предназначен для анализа проблем доставки и оценки эффективности перевозчиков.

Реализованные возможности:

- Анализ проблем по заказам через кастомный bubble chart

- Визуализация задержек, повреждений и объёма поставок в одном виджете

- Tooltip с расширенной информацией по показателям

- Одинарный клик — фильтрация заказов по перевозчику, двойной клик — переход на дашборд «Перевозчик» с наследованием глобальных фильтров

- Анализ средней стоимости доставки с интерактивной фильтрацией

- Кастомный виджет «Карта мира» с распределением заказов по странам

- Динамические tooltip’ы с разной логикой отображения данных

- Переход на дашборд «Основные показатели» с фильтрацией по выбранной стране

Перевозчик

Дашборд предназначен для оценки эффективности работы перевозчика и анализа рисков доставки.

Реализованные возможности:

- KPI по заказам и задержкам с динамикой к предыдущему периоду или сезону

- Визуализация процента задержанных заказов с пороговыми зонами риска

- Анализ повреждений заказов, товаров и затронутых клиентов

- Выявление стран с наибольшим количеством повреждений

- Кастомный комбинированный виджет (объём поставок и стоимость доставки)

- Многоуровневый drill-down по годам и месяцам

- Синхронизированный drill-down между виджетами

- Передача контекста в глобальные фильтры при взаимодействии с графиками
